# AI Search Engine Backend

## Quick Start

**Important: Run in WSL terminal for best performance**

```bash
# Open WSL terminal
wsl

# Navigate to backend
cd ~/projects/mini_search_engine/ai_search/backend

# Run the demo
python3 demo.py
```
